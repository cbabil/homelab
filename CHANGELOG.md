# Changelog

All notable changes to Tomo will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project structure with frontend, backend, and CLI
- User authentication system with JWT tokens
- Server management with SSH connectivity
- Applications deployment from marketplace
- Dashboard with system statistics
- Settings page with security and data retention options
- Access logs and audit logs pages
- Dark/light theme support
- Notification system
- Profile management

### Security
- Password strength validation
- Session management with automatic expiration
- Secure credential storage with encryption

## [0.1.0] - 2025-01-18

### Added
- Initial release of Tomo
- React frontend with MUI and Tailwind CSS
- Python MCP backend server
- SQLite database for data persistence
- CLI for administrative tasks
- Server CRUD operations
- SSH connection management
- Marketplace for application deployment
- User and admin account management
